I put the name of all the files in the fetchers array.
I create a random index and I use to choose a random image.
next I put the command:
neofetch with --ascii to print an ascii image and the dir of where the images are stored.

``` sh
fetchers=( "foto.txt" "foto2.txt" "pepe.txt" "pepe2.txt" "shreck.txt" "flip.txt" "bear.txt" "culo.txt" "nyan.txt" "bruh" "kekw" "trollge")

randomIndex=$((RANDOM % ${#fetchers[@]}))
neofetch --ascii <your dir>"${fetchers[randomIndex]}"

```

image that your script is at /home/<name/Documents/ascii/script.sh, you need to put on your rc the same path of your script.
Modify the script and puth the path of the directory that contains your imagines.
